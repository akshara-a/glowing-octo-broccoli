# College Website Using Bootstrap

## Project Overview

A simple responsive College Website created using HTML5,
CSS3, Bootstrap 5, and JavaScript.

## Features

- Responsive Navbar
- Hero Section
- About Section
- Courses Section
- Contact Form
- Footer
- Bootstrap Components
- Custom CSS Styling

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## How to Run

Open index.html in any browser.
