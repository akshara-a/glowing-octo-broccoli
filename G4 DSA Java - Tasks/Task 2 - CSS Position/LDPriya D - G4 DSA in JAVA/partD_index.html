Task 2 - Part D: Fixed Position

Fixed Position Example

This webpage demonstrates the use of fixed positioning in CSS. The header remains fixed at the top of the page, while the "Back to Top" button stays visible at the bottom-right corner even when the page is scrolled.

About Fixed Position

Fixed positioning places an element relative to the browser window. Unlike static or relative elements, a fixed element remains in the same location while the page is scrolling.

This technique is commonly used for navigation bars, floating action buttons, support chat icons, and notification panels.

Scroll further down to observe that the header and the "Back to Top" button remain in the same position.
