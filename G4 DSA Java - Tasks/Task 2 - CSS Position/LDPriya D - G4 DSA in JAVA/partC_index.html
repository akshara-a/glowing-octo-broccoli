Observation and Explanation

The child box is positioned at the top-right corner of the parent using position: absolute. When the parent has position: relative, the child is positioned relative to the parent container. If the parent does not have a positioning context (relative, absolute, fixed, or sticky), the child is positioned relative to the webpage instead of the parent.
