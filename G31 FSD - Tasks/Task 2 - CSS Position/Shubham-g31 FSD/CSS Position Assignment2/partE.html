Section 1 Content

Explain the difference between fixed and sticky.

Section 2 Content

POSITION : FIXED =- keeps an element permanently attached to the browser viewport. Once an element is fixed, it stays in the same location even when the user scrolls the page. It is removed from the normal document flow and remains visible at all times. This property is commonly used for floating chat buttons, back-to-top buttons, and fixed headers.

Section 3 Content

POSITION: STICKY =- keeps an element in the normal document flow until it reaches a specified point (like a threshold), at which point it becomes fixed. This property is useful for creating navigation menus that remain visible as the user scrolls down the page.
