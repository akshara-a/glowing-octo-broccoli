# FoodRush - Food Delivery Website

## Project Overview

FoodRush is a responsive food delivery website developed using HTML5, CSS3, Bootstrap 5, and JavaScript.

The project is designed to provide users with an easy way to browse food items, explore services, register, log in, and place food orders online.

This project is being developed as part of the CSS3 and Bootstrap UI Components Assignment.
